### 피보나치 수열

```python
def fib(n):
    _curr, next = 0, 1
    for  in range(n):
        _curr, _next = _next, _curr + _next
    return _curr
```





### 재귀함수로 표현

```python
def fib(n):
    if n == 0:
        return 0
    elif n == 1 or n == 2:
        return 1
    else:
        return fib(n - 1) + fib(n - 2)
```
